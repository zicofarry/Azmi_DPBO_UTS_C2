Buatlah desain relasi antar kelas di atas menggunakan konsep inner class (opsional), komposisi/asosiasi/agregasi (wajib), inheritance (wajib), interface/ kelas abstrak (wajib memakai salah satu) dan sertakan alasannya (bisa show off kemampuan inheritance, interface/kelas abstrak untuk nilai sempurna). Implementasikan semua kelas-kelas di atas sesuai desain yang dibuat.


Buatlah kelas Main yang menampilkan semua data dengan hubungannya (silakan mendesain hubungan antar kelas untuk ditampilkan, desain hubungan dapat dimasukkan dalam pdf). Main dapat menampilkan semua data hardcode dimana data hardcode dimasukkan dalam array objek. Data yang ada dalam array object minimal 2, kemudian Main dapat menerima masukan dari data utama (jelaskan pada desain, apa yang dapat dilakukan pencarian untuk menampilkan semua data yang terhubung) untuk selanjutnya dapat menampilkan data yang sesuai sebagai hasil pencarian.


Semua kelas memiliki metode get set dan komentar.


Semua peraturan pada mata kuliah pemrograman tetap berlaku kecuali yang ditimpa peraturan di soal ini (termasuk ada penulisan janji setia pada kejujuran).
